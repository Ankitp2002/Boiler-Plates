# Raw Docs Placeholder

This directory is intended to hold **source documents** for the LlamaIndex
KnowledgeBot project.

Examples of what you might store here:

- Markdown notes (e.g., your `LamaIndex.md` file exported as `.md`)
- Product documentation
- Interview preparation notes

The `scripts/ingest_docs.py` script is responsible for scanning this directory,
building a LlamaIndex index, and persisting it under `../index_placeholder.json`.
